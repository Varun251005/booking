# Booking Workspace

This project is now organized into separate folders:

- `frontend/` - React + Vite client app
- `backend/` - backend/server code

## Frontend

From the project root, run:

```bash
cd frontend
npm run dev
```

Other frontend commands:

```bash
cd frontend
npm run build
npm run preview
npm run lint
```
